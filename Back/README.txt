BDMMundial (versión estática)
================================

Contenido:
- HTML + CSS (sin PHP, sin JS)
- Assets (imagenes, videos, etc.)

Cómo verlo:
1) Abre `index.html` con doble click.
   - Si ves que algunas rutas absolutas no cargan, abre la carpeta con un servidor local (por ejemplo, VS Code Live Server).
2) Navega con el menú.

Notas:
- Se eliminó toda la lógica de PHP y JavaScript. Los botones/menús que dependían de JS (hamburguesa, modo oscuro, modales) quedan como diseño estático.
- Las secciones con datos dinámicos (usuarios, publicaciones, etc.) pueden mostrar placeholders o campos vacíos, pero conservan el layout.
